# Contributing

Thanks for your interest in contributing to the Maintainer Dashboard! See the [README](README.md) for setup instructions.

## Workflow

1. Fork the repository and create a feature branch from `main`
2. Make your changes
3. Ensure `npx tsc --noEmit` and `yarn build` pass
4. Open a pull request against `main`

## Code Standards

- TypeScript strict mode — no `any`, no unused variables
- Tailwind CSS for all styling
- Follow existing patterns and conventions in the codebase

## Reporting Issues

When filing an issue, please include:

- What you expected to happen
- What actually happened
- Steps to reproduce
- Browser and OS information (if relevant)
